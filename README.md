# Agent Readiness Scorecard Landing Page

Static landing page for Arpan Saxena's Agent Readiness Scorecard / AI Execution Audit offer.

## Strategy

- Page goal: convert enterprise leaders, founders, operators, and L&D heads into inquiries for the Agent Readiness Scorecard.
- Target visitor: people who know AI matters but do not yet have a structured workflow/adoption roadmap.
- Core promise: move from random AI usage to structured AI execution systems that save time and improve business execution.
- Primary CTA: Run the Agent Readiness Scorecard.
- Primary CTA: `mailto:connect@arpansaxena.com` and WhatsApp link from `arpansaxena.com`.

## Files

- `index.html` — landing page markup
- `styles.css` — responsive premium brutalist / systems-led styling
- `README.md` — project notes and deployment instructions

## Local preview

From this folder:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://127.0.0.1:8080
```

## Deploy to Vercel

Static site settings:

- Framework preset: Other
- Build command: none
- Output directory: `.`

## Replace before publishing

1. Replace the email/WhatsApp CTA with a calendar, Typeform, Tally, scorecard form, or CRM form if preferred.
2. Add approved case studies, logos, testimonials, screenshots, or before/after workflow proof.
3. Replace `og-image.svg` with a designed Open Graph image if needed.
