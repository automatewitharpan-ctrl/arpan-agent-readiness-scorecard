# Arpan Saxena Website Redesign

Premium static website / landing page for Arpan Saxena, redesigned using UI/UX Pro Max guidance and public information from:

- https://arpansaxena.com
- https://www.linkedin.com/in/arpan-saxena/

## Design direction

- Dark premium AI/automation aesthetic
- Bento card service architecture
- Large founder portrait in the hero
- Conversion-first CTA path: email, WhatsApp, LinkedIn, website
- Single primary business action: book/start an AI enablement conversation

## Files

- `index.html` — redesigned static page
- `styles.css` — responsive visual system
- `og-image.svg` — Open Graph image
- `assets/Gemini_Generated_Image_6sn60r6sn60r6sn6.png` — Arpan portrait from arpansaxena.com
- `assets/arpan-white-logo.png` — logo from arpansaxena.com
